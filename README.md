## Commit & Convention
- Commit
  - `feat : main 페이지 생성`
- Pull Request
  - `[feature/create-main-page] 메인페이지 생성`

## Branch Strategy
- main : 병합 
- docs : 문서 작업
- feature : 기능 개발
- test : 테스트
